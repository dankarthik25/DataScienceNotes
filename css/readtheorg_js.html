Function to Collapse TOC Elements on Click

function collapse_toc_elements_on_click(nav_li_a){
    /*
      When an `a' element in the TOC is clicked, its parent
      `li' element's active attribute is toggled.  This causes
      the element to toggle between minimized and maximized
      states.  The active attribute is documented in bootstrap.
      https://getbootstrap.com/docs/4.0/components/navbar/#nav
    */
    $(nav_li_a).parent().toggleClass("active");
}

Purpose: This function is designed to toggle the active class on the parent li element of a clicked a element within the Table of Contents (TOC). How it works:

  • It takes an a element as a parameter (nav_li_a).
  • It finds the parent li element of the clicked a element.
  • It toggles the active class on this li element, which affects its CSS styling, causing it to expand or collapse.

  • It binds a click event to each a element within the TOC (#text-table-of-contents a).
  • When an a element is clicked, it calls collapse_toc_elements_on_click.

Function to Replace Admonitions

$(function() {
    function replace_admonition (tag, readable) {
        $(`span.${tag}:not(#table-of-contents *)`) .parent().parent()
            .replaceWith(`<p id='${this.id}' class='admonition-title ${tag}'>${readable}</p>`);
        $(`div.${tag}`).before(`<p class='admonition-title ${tag}'>${readable}</p>`)
    }
    replace_admonition('note', 'Note');
    replace_admonition('seealso', 'See also');
    replace_admonition('warning', 'Warning');
    replace_admonition('caution', 'Caution');
    replace_admonition('attention', 'Attention');
    replace_admonition('tip', 'Tip');
    replace_admonition('important', 'Important');
    replace_admonition('hint', 'Hint');
    replace_admonition('error', 'Error');
    replace_admonition('danger', 'Danger');
});

Purpose: To replace certain HTML elements that have specific classes (like note, seealso, warning, etc.) with more readable formats. How it works:

  • The replace_admonition function takes two parameters: tag (the class of the element to be replaced) and readable (the text to replace it with).
  • It finds all span elements with the specified class, excluding those within the TOC.
  • It replaces these spans' grandparent elements with a paragraph element containing the readable text.
  • It also inserts a paragraph with the readable text before each div with the specified class.
  • The function is then called for each specified class (e.g., note, seealso, warning, etc.).

Document Ready Event 2

$( document ).ready(function() {
    // Shift nav in mobile when clicking the menu.
    $(document).on('click', "[data-toggle='wy-nav-top']", function() {
      $("[data-toggle='wy-nav-shift']").toggleClass("shift");
      $("[data-toggle='rst-versions']").toggleClass("shift");
    });
    // Close menu when you click a link.
    $(document).on('click', ".wy-menu-vertical .current ul li a", function() {
      $("[data-toggle='wy-nav-shift']").removeClass("shift");
      $("[data-toggle='rst-versions']").toggleClass("shift");
    });
    $(document).on('click', "[data-toggle='rst-current-version']", function() {
      $("[data-toggle='rst-versions']").toggleClass("shift-up");
    });
    // Make tables responsive
    $("table.docutils:not(.field-list)").wrap("<div class='wy-table-responsive'></div>");
});

Purpose: To add interactivity to the navigation and make tables responsive. How it works:

  • It sets up click event handlers to toggle CSS classes, adding responsive and interactive behavior to the navigation elements.
  • It ensures that clicking the navigation menu in mobile view toggles the shift class, affecting the layout.
  • It removes the shift class from the navigation when a menu link is clicked.
  • It adds a shift-up class when clicking the current version toggle.
  • It wraps all non-field list tables in a wy-table-responsive div, making tables responsive.

$( document ).ready(function() {
    $('#text-table-of-contents ul').first().addClass('nav');
                                        // ScrollSpy also requires that we use
                                        // a Bootstrap nav component.
    $('body').scrollspy({target: '#text-table-of-contents'});

    // DON'T add sticky table headers (Fix issue #69?)
    // $('table').stickyTableHeaders();

    // set the height of tableOfContents
    var $postamble = $('#postamble');
    var $tableOfContents = $('#table-of-contents');
    $tableOfContents.css({paddingBottom: $postamble.outerHeight()});

    // add TOC button
    var toggleSidebar = $('<div id="toggle-sidebar"><a href="#table-of-contents"><h2>Table of Contents</h2></a></div>');
    $('#content').prepend(toggleSidebar);

    // add close button when sidebar showed in mobile screen
    var closeBtn = $('<a class="close-sidebar" href="#">Close</a>');
    var tocTitle = $('#table-of-contents').find('h2');
    tocTitle.append(closeBtn);
});

Purpose: To set up the TOC and its interactivity. How it works:

  • It adds the nav class to the first ul element in the TOC, required for Bootstrap's ScrollSpy.
  • It initializes ScrollSpy on the body to highlight the TOC items based on scroll position.
  • It sets the padding of the TOC to match the height of the postamble element, ensuring proper spacing.
  • It adds a TOC button to toggle the sidebar, and a close button within the TOC for mobile screens.

SphinxRtdTheme Initialization

window.SphinxRtdTheme = (function (jquery) {
    var stickyNav = (function () {
        var navBar,
            win,
            stickyNavCssClass = 'stickynav',
            applyStickNav = function () {
                if (navBar.height() <= win.height()) {
                    navBar.addClass(stickyNavCssClass);
                } else {
                    navBar.removeClass(stickyNavCssClass);
                }
            },
            enable = function () {
                applyStickNav();
                win.on('resize', applyStickNav);
            },
            init = function () {
                navBar = jquery('nav.wy-nav-side:first');
                win    = jquery(window);
            };
        jquery(init);
        return {
            enable : enable
        };
    }());
    return {
        StickyNav : stickyNav
    };
}($));

Purpose: To handle sticky navigation. How it works:

  • It defines a stickyNav module that makes the navigation bar sticky if its height is less than or equal to the window height.
  • It initializes the navigation bar and window references.
  • It adds or removes the stickynav class based on the height condition.
  • It sets up an event listener to apply the sticky logic on window resize.

Overall, this JavaScript code enhances the interactivity and responsiveness of a webpage, particularly focusing on the Table of Contents and navigation elements, using jQuery for DOM manipulation and event handling.
