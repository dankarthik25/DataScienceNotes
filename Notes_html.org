** What is HTML 5 ?
HTML is latest evolution of standard that defines HTML. The term represents two different concept.

It is a new version of langugae HTML, with new elements, attributes and behaviors and a large set of technologies that alows the building of more diverse and powerfull Web sites and application.

** HTML ENTITIES

What are Semantic Elements?\\
A semantic element clearly describes its meaning to both the browser and the developer. \\
Examples of non-semantic elements: <div> and <span> - Tells nothing about its content.\\
Examples of semantic elements: <form>, <table>, and <article> - Clearly defines its content.\\
Semantic Elements in HTML\\
Many web sites contain HTML code like: <div id="nav"> <div class="header"> <div id="footer"> to indicate navigation, header, and footer.\\

** Block and inline Element : Div & Span
Div and Span are Block and inline respectively

*Block-level Elements:* \\
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.\\
A block-level element always takes up the full width available (stretches out to the left and right as far as it can).\\
Two commonly used block elements are: <p> and <div>.\\
https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div\\

*Inline Elements:* \\
An inline element does not start on a new line.\\
An inline element only takes up as much width as necessary. \\
This is a <span> element inside a paragraph.\\
https://developer.mozilla.org/en-US/docs/Web/HTML/Element/span\\
